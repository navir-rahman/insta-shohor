how does java script work?

js is an interpreted programming language, as it executes every code line by line. meaning it will read a line of code and then execute the line before going to the next line. it has 4 step to run a code.
1- parser: its the first step. it checks for code syntactic error.
2- AST: In this step js makes a data structure called ATC
3- Conversion to Machine code: at this step js engine converts cod into Machine code.
4- Machine code: at this step the converted code gets executed by system.

What does js event loop do?

JavaScript is single threaded programming language. so js runs a code line by line and it only excutes one line of code at time. and this is not effecient enough for the web. so js uses a tecnique which allows js to make an illution of being multi threaded. it stors data on heap memory, than makes a stack memeory. then makes function call stack. and finally event loopit pulls stuff out of the queue and places the code on function execution as soon as the function stack is empty.
